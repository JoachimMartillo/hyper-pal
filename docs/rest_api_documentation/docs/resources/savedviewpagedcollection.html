ADAM REST API Documentation

Resource: SavedViewPagedCollection

About this resource

Representation of a paged collection of SavedView items

Properties

Property Description Type
items One page of this paged collection, which is a subcollection of the complete collection. Collection of SavedView items
page The current page of this paged collection. int32
pageSize The current page size of this paged collection. int32
totalCount The total count of items in this paged collection, not just on the current page. int64

Relations

Method Name Select-key Resource type Description
GET self Provides a link to this resource.
GET first A link to the first page of this paged collection.
GET prev A link to the previous page of this paged collection.
GET next A link to the next page of this paged collection.
GET last A link to the last page of this paged collection.

Actions

Method Action Description Response codes
POST /savedviews create Creates a new saved view.
201 Request completed successfully. The header parameter 'Location' contains the location of the new record resource. The response body is a JSON object which contains the property "id" that corresponds to the newly created saved view resource id.
400 Bad request (in general)
403 Insufficient permissions
