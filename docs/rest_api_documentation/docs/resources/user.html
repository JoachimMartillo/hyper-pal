ADAM REST API Documentation

Resource: User

About this resource

Representation of a user.

Properties

Property Description Type
createdOn Gets the creation datetime in UTC time. datetime
email Gets or sets the email addres of this user. The email address it not required and has a maximum length of 254 characters. string
expirationDate Gets or sets the date on which this user account will expire. datetime
id Gets or sets the Id of this user. guid
languageForUIId Gets or sets the default language used by this user for the static text displayed on the user interfaces. guid
languageId Gets or sets the default language used by this user when accessing fields. guid
lastFailedLogOnDate Returns the last date on which someone tried to log on using this username and a wrong password. It returns null if no wrong password has ever been used or if the user has logged in successfully since. datetime
lastSuccessfulLogOnDate Returns the last UTC date on which someone successfully logged in using this user's account. It returns if no one has ever logged in using this account. datetime
lockedOut Gets or sets a bool that controls whether this user account is locked out or not. boolean
modifiedOn Gets the last modification datetime in UTC time. datetime
name Gets or sets the name of this user. This name is required, has to be unique, is not case sensitive and has a maximum length of 50 characters. string
secondLastSuccessfulLogOnDate Returns the second last date on which someone successfully logged in using this user's account. It returns if no one has ever logged in using this account. datetime
storageQuota Gets or sets the the maximum number of bytes that a user can own. The unit used is bytes. int64
storageUsed Gets the sum of all the filesizes of which this user is the owner. The unit used is bytes. int64
tag Gets or sets the value for the Tag of this object. The value of this property should be valid Xml. This property will not be returned by default. In order to include the property in the response, add a header with the name 'select-user' and the value 'Tag' to your request. string

Relations

Method Name Select-key Resource type Description
GET self Provides a link to this resource.
GET fields select-User: fields FieldCollection Gets a collection of all the fields of this user.

GET image select-User: image Image Gets an object allowing you to control the picture of this user.
GET modifiedby select-User: modifiedby User Gets or sets the user that last modified this resource.
