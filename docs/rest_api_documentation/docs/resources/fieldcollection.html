ADAM REST API Documentation

Resource: FieldCollection

About this resource

Representation of a non-paged collection of fields

Properties

Property Description Type
items A collection of resource items. Collection of items containing any combination of following types:
  • ClassificationListField
  • DynamicOptionListField
  • JsonField
  • TextListField
  • DateField
  • DateTimeField
  • DurationField
  • HtmlField
  • LanguageListField
  • MultiLineTextField
  • NumericField
  • OptionListField
  • RecordLinkField
  • RecordListField
  • SingleLineTextField
  • TimeField
  • UserGroupListField
  • UserListField
