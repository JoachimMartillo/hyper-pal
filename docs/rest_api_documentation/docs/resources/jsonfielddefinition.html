ADAM REST API Documentation

Resource: JsonFieldDefinition

About this resource

Representation of the definition of a JsonField

Properties

Property Description Type
createdOn Created on the specified date in UTC time. datetime
dataType Gets the data type of this field. string (values: none, singlelinetext, multilinetext, numeric, datetime, date, time, optionlist, duration, userlist, usergrouplist, html, json, classificationlist, recordlist, recordlink, languagelist, dynamicoptionlist, textlist)
enabledLanguages Enabled languages. Collection of guid items
id Gets the identifier for this field. guid
indexed Gets the indicator if the field is indexed. boolean
inlineStyle Gets a inline css style for this field. string
isReadOnly Indicates whether this field is read only. boolean
isRequired Indicates whether this field is required. boolean
isUniqueIdentifier Indicates whether this field is a unique identifier. boolean
label Gets the label of this field. string
languageMode Gets the language mode of the field. string (values: single, multiple)
memberships Gets the memberships of a field. Collection of guid items
modifiedOn Modified on the specified date in UTC time. datetime
name Gets the name for this field. string
schemaUpdateRequired Gets the indicator if the schema update is required. boolean
scope Gets the scope. string (values: recordcontentglobal, recordcontentclassdependent, recordcontentfloating, classificationprofileclassdependent, userglobal, filefiletypedependent, filefloating, indexertaskglobal, fileglobal, classificationprofilefloating, recordcontentcontenttypedependent, classificationprofilecontenttypedependent)
scopeCategory Gets the scope category. string (values: record, file, classification, user, indexertask)
sortIndex Gets the sorting index of this field. int32
tag Gets a tag for this field containing extra information. This property will not be returned by default. In order to include the property in the response, add a header with the name 'select-jsonfielddefinition' and the value 'Tag' to your request. string
