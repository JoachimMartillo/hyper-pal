ADAM REST API Documentation

Entity: DynamicOptionListFieldValues

About this entity

The collection of language-specific values stored in the dynamic option list field.

Properties

Property Description Type
languageId The language id. guid
modifiedOn Contains the date/time when the field is last modified. datetime
readOnly Tells if the field is read-only. boolean
values Language-specific collection of values of a dynamic option list. Collection of key-value pairs with string keys and string values
