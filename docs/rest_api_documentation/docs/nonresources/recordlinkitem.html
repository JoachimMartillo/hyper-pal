ADAM REST API Documentation

Entity: RecordLinkItem

About this entity

Record link item.

Properties

Property Description Type
modifiedOn Contains the date/time when the item was last modified. datetime
recordId The record Id (represented as a GUID value). guid
tag The tag. This property will not be returned by default. In order to include the property in the response, add a header with the name 'select-field' and the value 'Tag' to your request. string
