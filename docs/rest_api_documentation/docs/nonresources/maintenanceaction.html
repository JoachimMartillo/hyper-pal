ADAM REST API Documentation

Entity: MaintenanceAction

About this entity

Representation of an action that can be applied on targets of a MaintenanceJob.

Properties

Property Description Type
action Type of the action. string (values: appendtextfield, assignwatermarkrecord, classificationcopy, classificationmove, classifyrecord, delete, forcedirty, overwritefield, recordsave, refreshfilerecord, refreshfiletyperecord, replacetextfield, resettodefaultfield, touch, unclassifyrecord, updatedynamicoptionlistvalue, changecontenttype, resizeimage, transcodevideo)
description Short description of the action. string
impact Estimated number of milliseconds it will take for this action to execute. int64
label Name of the action. string
parameters Action-specific parameters. Dictionary of pairs with string keys and object values
