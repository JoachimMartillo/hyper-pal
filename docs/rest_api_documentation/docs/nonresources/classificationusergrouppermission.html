ADAM REST API Documentation

Entity: ClassificationUserGroupPermission

About this entity

Representation of a classification access right for a user group.
The following rights you can only apply to classification tree: classifydenymodify, readdenyclassify, classify.

Properties

Property Description Type
accessRight Gets the access right of this permission. string (values: noaccess, read, classify, modify, delete, fullcontrol, deletedenyfullcontrol, modifydenydelete, classifydenymodify, readdenyclassify, denyread, inherit)
userGroupId Gets the id of the user group this permission applies too. guid
